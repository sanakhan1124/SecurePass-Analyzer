# 🔐 SecurePass Analyzer

A Python-based Cyber Security Password Strength Checker that evaluates password security using Regex and cybersecurity best practices.

---

## 🚀 Features

✅ Detects Weak, Medium, and Strong passwords  
✅ Checks password length  
✅ Detects uppercase and lowercase characters  
✅ Detects numbers and special symbols  
✅ Provides password improvement suggestions  
✅ Colored terminal output using Colorama  
✅ Beginner-friendly cyber security project

---

## 🛠️ Technologies Used

- Python
- Regex (`re`)
- Colorama

---

## 📌 How It Works

The program analyzes passwords based on:

- Minimum length
- Uppercase letters
- Lowercase letters
- Numbers
- Special characters

Based on the score, the password is classified as:

- ❌ Weak
- ⚠️ Medium
- ✅ Strong

The system also provides suggestions to improve password security.

---

## ▶️ How to Run

1. Install Python
2. Install Colorama

```bash
pip install colorama
```

3. Run the project

```bash
python main.py
```
